A key part of my role as co-founder of Zippy Brain is to build and manage websites for our various properties, though I also do it for fun. This page showcases some of the sites I've worked on.

  • WealthSim App Promo Site

    Live at WealthSim.com

    I built this site to start building pre-release buzz for WealthSim, our personal finance RPG. It's built to match the trend of single page app marketing websites, and it uses jQuery for custom animations and interactivity.

    • Built with Bootstrap, Less, Jekyll, and jQuery
    • Hosted on GitHub with GitHub Pages
    • Analyzed by Google Analytics

  • Personal Portfolio Site

    You're looking at it

    After working on a small team for a while, I realized that I needed a place to show what kind of work I've been doing. This site serves both as a portfolio site and as a place for me to post documentation on how I've solved business or analytical challenges. It will always be a work in progress.

    • Built with Bootstrap, Less, Jekyll, and jQuery
    • Syntax highlighting by Pygments
    • Hosted on GitHub with GitHub Pages
    • Analyzed by Google Analytics

  • Exologistics Marketing Site

    Live at SantaMonicaExologistics.com

    Santa Monica Exologistics is a fictional company and nascent idea for a simulation or management game. I went ahead and registered the domain, then built the company marketing website as an exercise.

    • Built with Bootstrap and Less

  • More coming soon.
